# E-commerce PHP & MySQL

Final course project demonstrating full-stack web development skills.

## Technologies

- HTML5
- CSS3
- Bootstrap
- JavaScript
- PHP
- MySQL

## Features

- User registration and authentication
- Product catalog
- Shopping cart
- Checkout system
- Admin panel (users, products, and orders management)
- Image upload system

## Project Structure

/actions
/admin
/assets
/includes
/sql
/uploads

## Installation

1. Import the database from the `/sql` folder
2. Configure the MySQL connection in `/includes`
3. Run the project using XAMPP or any Apache server
